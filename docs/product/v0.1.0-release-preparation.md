- deterministic evidence generation from provisional session-scoped terms;
- explicit human review authority over every materialized change;
- reviewed output derived from source plus accepted decisions only;
- deterministic decision log and session summary artifacts;
- strict skeleton-compatible `compare` and `evaluate` correspondence support for calibration workflows.
